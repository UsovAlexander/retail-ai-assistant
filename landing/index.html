Execution vs Result

Две метрики

Execution — SQL исполнился. Result — результат совпал с эталоном. Строгая: лишняя колонка = провал.

Retry работает

≤3 попытки

Ошибка ClickHouse уходит обратно в модель текстом. Внешней 70B ретраи не нужны — 1.00 попытки.

Автофолбэк

auto = Groq → local

В проде: сначала внешняя, при лимите/сбое — мгновенно локальная. Проверено на живом 429 от Groq.
